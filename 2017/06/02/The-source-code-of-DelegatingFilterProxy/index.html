з”ЁиҝҮSpring Securityзҡ„дәәйғҪдјҡзҶҹжӮүд»ҘдёҠд»Јз ҒпјҢдёҖиҲ¬ж•ҷзЁӢдёҠйғҪдјҡиҜҙжҳҺпјҢдҪҝз”ЁSpring Securityе°ұйңҖиҰҒеңЁweb.xmlдёӯжҢҮе®ҡд»ҘдёҠд»Јз ҒгҖӮд»ҺиҝҷдёӘй…ҚзҪ®дёӯпјҢеҸҜиғҪдјҡз»ҷжҲ‘д»¬йҖ жҲҗдёҖдёӘй”ҷи§үпјҢд»ҘдёәDelegatingFilterProxyзұ»е°ұжҳҜspringSecurityзҡ„е…ҘеҸЈпјҢдҪҶе…¶е®һиҝҷдёӘзұ»дҪҚдәҺspring-webиҝҷдёӘjarдёӢйқўпјҢиҜҙжҳҺиҝҷдёӘзұ»жң¬иә«жҳҜе’ҢspringSecurityж— е…ігҖӮ

2. зңҹзӣё

DelegatingFilterProxyзұ»з»§жүҝдәҺжҠҪиұЎзұ»GenericFilterBean,й—ҙжҺҘең°implement дәҶjavax.servlet.FilterжҺҘеҸЈпјҢServletе®№еҷЁеңЁеҗҜеҠЁж—¶пјҢйҰ–е…Ҳдјҡи°ғз”ЁFilterзҡ„initж–№жі•,GenericFilterBeanзҡ„дҪңз”Ёдё»иҰҒжҳҜеҸҜд»ҘжҠҠFilterзҡ„еҲқе§ӢеҢ–еҸӮж•°иҮӘеҠЁең°setеҲ°з»§жүҝдәҺGenericFilterBeanзұ»зҡ„FilterдёӯеҺ»гҖӮеңЁе…¶initж–№жі•зҡ„еҰӮдёӢд»Јз Ғе°ұжҳҜеҒҡдәҶиҝҷдёӘдәӢпјҡ

1protected Filter initDelegate(WebApplicationContext wac) throws ServletException {

йҖҡиҝҮи·ҹиёӘд»Јз ҒпјҢеҸ‘зҺ°еҸ–еҮәзҡ„beanжҳҜorg.springframework.security.FilterChainProxyпјҢиҜҘзұ»д№ҹжҳҜз»§жүҝдәҺGenericFilterBean,еҸ–еҮәbeanеҗҺпјҢеҲӨж–ӯtargetFilterLifecycleеұһжҖ§жҳҜfalseиҝҳжҳҜtrue,еҶіе®ҡжҳҜеҗҰи°ғз”ЁиҜҘзұ»зҡ„initж–№жі•гҖӮиҝҷдёӘFilterChainProxy beanе®һдҫӢжңҖз»Ҳиў«дҝқеӯҳеңЁDelegatingFilterProxyзұ»зҡ„delegateеұһжҖ§йҮҢ,

д»ҘдёҠе°ұжҳҜDelegatingFilterProxyзұ»зҡ„дёҖдәӣеҶ…йғЁиҝҗиЎҢжңәеҲ¶пјҢе…¶е®һдё»иҰҒдҪңз”Ёе°ұжҳҜдёҖдёӘд»ЈзҗҶжЁЎејҸзҡ„еә”з”Ё,еҸҜд»ҘжҠҠservlet е®№еҷЁдёӯзҡ„filterеҗҢspringе®№еҷЁдёӯзҡ„beanе…іиҒ”иө·жқҘгҖӮ

жӯӨеӨ–иҝҳиҰҒжіЁж„ҸдёҖдёӘDelegatingFilterProxyзҡ„дёҖдёӘеҲқе§ӢеҢ–еҸӮж•°пјҡtargetFilterLifecycle ,е…¶й»ҳи®ӨеҖјдёәfalse гҖӮ дҪҶеҰӮжһңиў«е…¶д»ЈзҗҶзҡ„filterзҡ„init()ж–№жі•е’Ңdestry()ж–№жі•йңҖиҰҒиў«и°ғз”Ёж—¶пјҢйңҖиҰҒи®ҫзҪ®targetFilterLifecycleдёәtrueгҖӮе…·дҪ“еҸҜи§ҒDelegatingFilterProxyдёӯзҡ„еҰӮдёӢд»Јз Ғпјҡ

3. з»“и®ә

DelegatingFilterProxyжҳҜSpring-webдёӯзҡ„дёҖдёӘйҖҡз”ЁFilterпјҢе®ғеҸҜд»ҘиҫҫеҲ°еҰӮдёӢзӣ®зҡ„пјҢи§ЈеҶіеҰӮдёӢй—®йўҳпјҡ йҖҡиҝҮDelegateжЁЎејҸпјҢе°ҶFilterзҡ„еҠҹиғҪиҪ¬з§»еҲ°дёҖдёӘSpringз®ЎзҗҶзҡ„BeanдёҠеҺ»пјҢиҝҷж ·иҝҷдёӘBeanе°ұеҸҜд»Ҙе®һзҺ°FilterеҠҹиғҪпјҢиҖҢдё”еҸҲдә«еҸ—дәҶSpringеёҰжқҘзҡ„еҘҪеӨ„гҖӮ еҸҰеӨ–пјҡиҝҷдёӘBeanйңҖиҰҒе®һзҺ°FilterжҺҘеҸЈгҖӮ
